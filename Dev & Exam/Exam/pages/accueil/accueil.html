DÉFINITION

Par définition, Un site web, site Web ou simplement site, est un ensemble de pages web et de ressources reliées par des hyperliens, défini et accessible par une adresse web. Un site est développé à l'aide de langages de programmation web, puis hébergé sur un serveur web accessible via le réseau mondial Internet, un intranet local, ou n'importe quel autre réseau, tel que le réseau Tor. L'ensemble des sites web disponibles sur le net constituent le World Wide Web.
Exemples :
MIT (cours.mit.sn)
Google (www.google.com)
Facebook (www.facebook.com)

ACCÉS Á UN SITE WEB ?

L’accès à un site se fait généralement à travers un navigateur. Le rendu peut différer d’un navigateur à un autre. Les navigateurs les plus connus sont : Google Chrome, Mozilla Firefox, Safari, Internet Explorer, Opéra, etc.

CREATION DE SITE WEB

Sites statiques Vs sites dynamiques

  1. Les sites web statiques

    2. Ils ont le mécanisme de fonctionnement le plus simple. Les URL correspondent à des fichiers renvoyés par le serveur web. Le contenu des fichiers demandés est restitué tel quel : aucune interprétation. Pour changer le rendu d'une page, il est nécessaire de changer son code source. Les visiteurs peuvent seulement voir le contenu du site mais pas y contribuer.
    Technologies utilisées :
    HTML : Langage de description du contenu de la page
    CSS : Mise en page / mise en forme du contenu de la page
    Javascript : Rendre le site interactif, exécuté du côté du navigateur

  2. Les sites web dynamiques

    3. Peuvent offrir les mêmes services que les sites statiques. Offrent un contenu issu de l’interprétation des pages demandées. Le contenu des pages est généré par des programmes qui tourne sur le serveur. Peuvent interagir avec des sources de données externes (bases de données, etc.) les visiteurs peuvent contribuer à son contenu (commentaire, forum, wiki, etc.
    Technologies utilisées (en plus de celles utilisées par les sites statiques) :
    PHP / Asp .NET / Java EE, etc & SGBD : MySQL / PostegreSQL / Oracle.

Outils de développement

  • Editeur de texte (SublimeText)
  • Serveur web (Apache)
  • Un interpréteur PHP
